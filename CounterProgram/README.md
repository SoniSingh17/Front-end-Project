# Counter Web Project

## 📁 Files
- index.html: UI structure
- style.css: Styling
- script.js: Counter logic

## 🚀 How to Run
Open index.html in any browser (double-click or run 'start index.html' from Git Bash)

## 📌 Features
- Display counter
- Increase / Reset / Decrease buttons
- Smooth hover effect
